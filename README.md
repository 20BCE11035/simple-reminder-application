# simple-reminder-application
Create a simple reminder application. A person has to be reminded on the below activities on daily basis in a span of 24 hour  
[Day of the week] 
Wake up 
Go to gym 
Breakfast
Meetings
Lunch
Quick nap 
Go to library
Dinner
Go to sleep.
